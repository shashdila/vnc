
Install VNC with Firefox
=========================================

1. wget https://raw.githubusercontent.com/shashdila/vnc/main/VNC.sh
2. chmod +x VNC.sh
3. ./VNC.sh

VNC Troubleshooting
=========================================

Q: VNC is stuck?

A: Run the followin commands;
- reboot
- vncserver :1 -geometry 1366x768
